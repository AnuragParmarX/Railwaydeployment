About The Project

Helping growers identify issues earlier

This Plant Disease Detection interface combines a clean startup-style UI with an AI prediction workflow. Users can upload a leaf image, receive a likely disease diagnosis from the backend model, and review practical treatment guidance stored directly in the frontend for a lightweight, database-free experience.
